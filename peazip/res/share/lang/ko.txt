아카이브 추출
시스템에서 
- 아카이브를 마우스 오른쪽 버튼으로 클릭하고 시스템의 상황에 맞는 메뉴에서 "여기에 추출" 또는 "여기에 추출(새 폴더에서)"을 클릭하여 추가 상호 작용 없이 추출합니다.
- 그렇지 않으면 "추출..." 더 많은 옵션을 갖기위한 메뉴 항목 : 출력 경로, 비밀번호, 새 디렉토리로 파일 추출, 건너뛰기, 이름 바꾸기 또는 기존 파일 덮어쓰기 선택 등.

PeaZip에서 아카이브를 열거나(예: 더블 클릭) PeaZip의 창 또는 아이콘에서 아카이브를 드래그합니다.
- 도구 모음 또는 상황에 맞는 메뉴에서 "추출"을 클릭하여 모든 옵션 (출력 경로, 비밀번호, 명명 정책, 새 디렉토리로 추출 등)을 설정할 수 있는 확인 대화상자에서 선택한 개체만 추출합니다.
- 파일 및 폴더를 원하는 대상으로 드래그하면 선택한 콘텐츠만 추출됩니다(자세한 내용은 PeaZip의 드래그 앤 드롭에 대한 장을 참조).
- 도구 모음의 "추출" 오른쪽에 있는 빠른 추출 드롭다운 버튼을 클릭하면 전체 아카이브를 가장 일반적인 대상으로 직접 추출하고(확인 대화상자 건너뛰기) 가장 중요한 옵션을 설정할 수 있습니다; 키보드 단축키: Ctrl+E(또는 F12)는 전체 아카이브를 추출하고 출력 디렉토리를 요청합니다; Ctrl+Alt+E는 현재 폴더에서 아카이브를 추출합니다; Ctrl+Shift+E는 데스크탑으로 추출합니다; Ctrl+Alt+Shift+E는 문서로 추출합니다; Ctrl+1..8는 북마크 1..8의 경로로 추출(정의된 경우)

PeaZip을 열고 하나 이상의 아카이브를 선택한 다음 도구 모음 또는 상황에 맞는 메뉴에서 "추출"을 클릭하거나 이전 지점에서 설명한 대로 빠른 추출 대상을 사용하십시오.

아카이브에서 선택한 개체만 추출
PeaZip에서 아카이브를 열거나(예: 더블 클릭) PeaZip의 창 또는 아이콘에서 아카이브를 드래그합니다.
- 툴바에서 "추출" 클릭: 선택한 항목만 추출됩니다.
- 파일 및 폴더를 원하는 위치로 드래그하면 선택한 콘텐츠만 추출됩니다.
- 마우스 오른쪽 버튼을 클릭하고 상황에 맞는 메뉴의 "추출"그룹에서 "선택 항목 추출"(선택한 항목 만 추출) 또는 "보이는 항목 추출"(현재 디렉토리 또는 현재 검색 필터의 내용을 추출)을 클릭합니다.

비밀번호를 입력하여 아카이브 찾아보기 또는 추출
자물쇠 아이콘을 클릭하여 비밀번호와 선택적으로 키 파일을 입력하십시오
이 아이콘은 PeaZip의 파일 브라우저 상태 표시줄과 아카이브 추출 인터페이스 모두에 표시됩니다. 비밀번호가 설정되면 아이콘의 색상이 변경됩니다.

아카이브 생성
시스템에서
- 아카이브 개체를 마우스 오른쪽 버튼으로 클릭하고 상황에 맞는 메뉴 또는 보내기 메뉴에서 "아카이브에 추가"를 클릭합니다. 아카이브 생성 확인 대화 상자가 열리고 "고급" 탭에서 더 많은 옵션을 사용할 수 있습니다. 확인을 클릭하여 아카이브를 만듭니다.
- 또는 파일/폴더를 PeaZip의 창이나 바로 가기로 드래그하십시오. 동일한 아카이브 생성 인터페이스가 표시됩니다.

PeaZip에서
- 아카이브할 개체를 선택하고 "추가" 버튼을 클릭하십시오. 동일한 아카이브 생성 인터페이스가 표시됩니다.

파일/폴더를 추가하여 기존 아카이브 업데이트
PeaZip에서 아카이브를 연 다음(예 : 더블 클릭하거나 PeaZip의 창 또는 아이콘에서 아카이브를 드래그) 아카이브에 추가할 파일 및 폴더를 드래그하십시오(또는 "추가"버튼을 클릭하고 응용 프로그램의 상황에 맞는 메뉴를 사용하여 아카이브에 개체를 추가하십시오).
아카이브 생성 확인 대화 상자가 열리고 확인을 클릭하여 기존 아카이브를 업데이트합니다.

더 작은 파일로 분할된 아카이브 생성
이전 항목에서 설명한 대로 아카이브를 생성하는 동안 "단일 볼륨" 드롭다운 메뉴를 클릭하여 아카이브가 분할될 출력 파일(볼륨)의 크기를 선택합니다.
가장 일반적인 아카이브 유형은 이 옵션을 지원합니다.

암호화된 아카이브 생성
비밀번호와 선택적으로 키 파일을 입력하려면 자물쇠 아이콘을 클릭하십시오; 아이콘은 PeaZip의 파일 브라우저 상태 표시줄과 아카이브 작성 인터페이스에 모두에 표시됩니다.
아카이브 확인 "파일 이름도 암호화"에 포함된 파일 및 디렉토리의 이름을 숨기려면 7Z 및 ARC와 같이 이 기능을 지원하는 형식으로 아카이브가 생성되는 경우에만 적용됩니다.
아카이브 생성 인터페이스에는 자물쇠 아이콘과 함께 암호화가 설정되어 있는지, 현재 아카이브 형식이 암호화를 지원하는지 시각적으로 알려주는 경고가 있습니다.

개별 아카이브 생성
아카이브할 개체를 추가하고(PeaZip의 "추가" 단추를 사용하거나 시스템의 상황에 맞는 메뉴 또는 보내기 메뉴에서) "확인"을 눌러 확인하기 전에 "개별 아카이브에 추가" 옵션을 선택하십시오.

아카이브 변환
PeaZip에서 변환할 아카이브를 선택하고 도구 모음 또는 상황에 맞는 메뉴에서 "변환"을 클릭하면 아카이브가 아닌 파일과 폴더도 추가할 수 있습니다. 차이점은 압축 단계 전에 아카이브에서 추출됩니다.
"기존 아카이브 변환"스위치를 "개별 아카이브에 추가"(기본값)와 함께 사용하면 스위치 아카이브 변환없이 나열된 아카이브의 대량 변환을 수행하므로 입력 데이터를 단일 아카이브에 통합하여 압축 효율을 향상시킬 수 있습니다. 압축되지 않은 형식의 원본 데이터를 다시 압축할 수 있습니다.

지정된 형식과 선택한 압축 수준으로 아카이브를 직접 생성
옵션 메뉴에서 "시스템 통합"절차를 사용하면 상황에 맞는 메뉴 항목을 활성화하여 선택한 파일/폴더를 ZIP, 7Z 또는 자체 추출 아카이브에 직접 추가 할 수 있습니다.
Zip 및 7Z 형식의 경우 선택한 형식의 기본 압축 수준을 우회하여 상황에 맞는 메뉴 항목을 가장 빠름, 일반 또는 최강 수준으로 압축 할 수도 있습니다.

비밀번호 관리자
비밀번호 관리자는 메인 메뉴, 도구 > 비밀번호 관리자 또는 비밀번호 양식에서 비밀번호 필드 왼쪽의 드롭 다운 메뉴에서 액세스 할 수 있습니다.
저장된 비밀번호는 비밀번호 양식으로 제출된 비밀번호 왼쪽의 드롭다운 메뉴에서 선택하거나 비밀번호 관리자에서 복사하여 다른 응용프로그램에서 사용할 수 있습니다.
비밀번호 관리자에 대한 마스터 비밀번호가 설정되지 않은 경우 인증 프롬프트 없이 비밀번호 목록에 액세스할 수 있습니다. 그렇지 않으면 사용자가 올바르게 인증할 때까지 비밀번호가 표시되지 않습니다.

응용 프로그램 설정
옵션 > 언어(및 옵션 > 설정, 첫 번째 탭)에서 응용 프로그램 언어를 설정할 수 있습니다.
옵션 > 설정을 통해 응용 프로그램의 설정을 편집할 수 있습니다.
옵션 > 테마를 통해 응용 프로그램의 아이콘과 모양을 변경할 수 있습니다.
옵션 > 시스템 통합은 파일 연결, 상황에 맞는 메뉴, 보내기 메뉴의 항목을 설정하는 마법사를 시작합니다(Windows의 경우).
구성 메뉴에는 도구 모음, 주소 표시줄, 사이드 바 등과 같은 브라우저의 기능을 사용자 정의, 활성화 또는 비활성화하는 항목이 포함되어 있습니다.

PEAZIP 정보

PeaZip은 오픈소스 파일 및 압축 매니저입니다: 32/64비트 윈도우즈(9x, 2k, XP, Vista)와 리눅스(PeaZip은 데스트탑 중립적인 프로그램입니다)를 지원하는 크로스 플랫폼 소프트웨어이며, 포터블과 설치 버전으로 사용할 수 있습니다.

PeaZip은 아카이브의 항목에 대해 강력한 다중 검색 필터를 지원합니다; 여러개의 아카이브를 한꺼번에 생성하고 추출할 수 있습니다; 자동 풀림 아카이브를 생성할 수 있습니다; 명령행으로 작업 정의 파일을 내보낼 수 있습니다; 아카이브 및 추출 레이아웃을 저장할 수 있습니다; 아카이브 및 폴더 북마크를 지원합니다; 아카이브나 압축이 안된 파일을 사용자가 원하는 프로그램으로 스캔하고 여는 등의 기능을 제공합니다...

다른 기능: 강력한 암호화, 안전한 파일 복사, 파일을 분할/합치기, 안전한 데이터 삭제, 비교, 검사합과 해쉬 파일, 시스템 벤치마크, 무작위 암호 및 키 파일 생성.
